Joke  
=====
The web cosole for [godns](https://github.com/kenshinx/godns)

### Running

```
$ go get github.com/kenshinx/joke
$ go get github.com/astaxie/bee

$ cd $GOPATH/src/github.com/kenshinx/joke

$ bee run joke  

```

### Configuration

`conf/app.conf` 


### Auth

Support http basic auth
